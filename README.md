# My First ADR
ADR(Architecture Decision Record)을 손쉽게 관리할 수 있는 웹 어플리케이션입니다.

## Dependencies
* Java 21
* SpringBoot 4.0.0
* MongoDB
* MySQL

## Features
* 프로젝트를 생성하여 ADR들을 그룹으로 묶을 수 있다. (미구현)
* ADR을 생성/수정/삭제하고 생성된 ADR을 조회할 수 있다. (미구현)
* ADR에 사용되는 템플릿을 커스터마이징할 수 있다. (미구현)
* 로그인을 꼭 하지 않아도 게스트로 접속하여 사용할 수 있으나, 쿠키를 삭제하면 데이터는 휘발된다. (미구현)
* 로그인을 하게 되면 데이터를 영구히 저장할 수 있다. (미구현)
* 게스트 상태에서 로그인하면 기존에 작성한 데이터를 계정에 연동할 수 있다. (미구현)
* 사용자 이름을 변경할 수 있다. (미구현)
* 다른 사용자를 초대하여 프로젝트 혹은 ADR을 공유할 수 있다. (미구현)
* 조직을 생성하여 조직 내 사용자를 추가하고 조직 단위로 프로젝트 및 ADR을 관리할 수 있다. (미구현)
* ADR을 Markdown 형식으로 변환할 수 있다. (미구현)
